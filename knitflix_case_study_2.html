Knitflix - Case Study

Overview

Knitflix is a single-page web app where users can explore movies, directors, and genres — and build their own list of favorites. It’s fast, responsive, and built with the MERN stack (MongoDB, Express, React, Node.js).

Purpose and Context

I built Knitflix as part of of my Full Stack Web Development Course through Career Foundry to add to my professional portflio.

  • Create several views (like movie details, profile, or favorites) using React
  • Demonstrate mastery of full-stack Javascript development.

Objective

Build the server-side and client-side for a movies web application. The movies application will allow users to access information about various movies, directors and genres. User will be able to sign up, update their profile and create a list of their favorite movies.

Clien-Side Overview

This is the front-end of Knitflix — the part users interact with. It’s built as a Single Page Application (SPA) using React, and connects to the server via a REST API. Here’s how I built it, step by step:

Step 1: Project Setup

I started by setting up React, Node.js, npm, and Parcel. Since React is component-based, I brought in Parcel early to streamline the build process. Parcel handles transpiling — converting JSX (React’s syntax) into regular JavaScript so browsers can understand it.

Step 2: Building the Views

I created a separate React Component for each major view in the app:

  • Login View
  • Signup View
  • Movie View - displays movie details
  • Profile View - shows user info and favorites

Each component connects to the backend using React Hooks to fetch and manage data from the API.

Step 3: Forms, Favorites and Validation

  • Built React forms for login and signup, with validation to ensure clean input.
  • Added authentication logic so user login info is stored in localStorage until logout.
  • Displayed each movie as a card, making it easy to browse.
  • Implemented navigation between views using React Router for smooth transitions.

    • Step 4: Styling and Deployment

  • Styled the app using React Bootstrap, including a grid system to lay out movie cards responsively.
  • Hosted the final app on Netlify for fast, reliable deployment.
